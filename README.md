## 🚀 Getting Started

### Python Projects
Make sure Python is installed, then:
```bash
pip install pygame  # only needed for Snake Game
python project-name.py
```

### C# Projects
Make sure .NET is installed, then:
```bash
dotnet run
```

### JavaScript Projects
Make sure Node.js is installed, then:
```bash
node filename.js
```
